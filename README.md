# אפליקציית נגן RTSP - אתגר התאמה

אפליקציית דסקטופ ל-Windows המאפשרת להזין כתובת RTSP ולנגן וידאו בזרם חי בצורה יציבה.

## דרישות מוקדמות
1. **Python 3.10+** מותקן על המערכת.
2. **VLC Media Player** מותקן על ה-Windows (חובה עבור הספרייה `python-vlc`). ניתן להוריד מ-[אתר VLC הרשמי](https://www.videolan.org/vlc/).

## הוראות התקנה והרצה
1. פתח טרמינל בתיקיית הפרויקט.
2. צור סביבה וירטואלית (מומלץ):
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\activate
   ```
3. התקן את הספריות הנדרשות:
   ```powershell
   pip install -r requirements.txt
   ```
4. הרץ את האפליקציה:
   ```powershell
   python main.py
   ```

## תכונות
- הזנת כתובת RTSP (כולל תמיכה בשם משתמש וסיסמה).
- כפתור נגינה וכפתור עצירה.
- תצוגת וידאו חלקה בתוך החלון.
- הודעות סטטוס ושגיאה מפורטות בעברית.
- יציבות משופרת באמצעות שימוש ב-RTP over TCP.

## טיפול בשגיאות
- האפליקציה תציג הודעה באדום במקרה של כתובת לא תקינה, חוסר חיבור לרשת או זרם וידאו שאינו זמין.
